
1. Create Project Directory



2. Create Docker Compose File/
services:
  jenkins:
    image: jenkins/jenkins:lts
    container_name: jenkins
    restart: unless-stopped
    ports:
      - "8080:8080"
      - "50000:50000"
    volumes:
      - jenkins_home:/var/jenkins_home
      - /var/run/docker.sock:/var/run/docker.sock

volumes:
  jenkins_home: {}



3. Start Jenkins Container
powershell
docker compose up -d

Phase 2: Jenkins Initial Setup
4. Get Initial Admin Password
powershell
docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword
5. Access Jenkins Web UI
Opened browser: http://localhost:8080

Entered the initial admin password

Installed suggested plugins

Created admin user account

Phase 3: Troubleshooting (Password Reset)
6. Reset Forgotten Password
When you forgot the password after restart:

powershell
# Copy config file from container
docker cp jenkins:/var/jenkins_home/config.xml .\config.xml

# Edit config.xml - changed:
# <useSecurity>true</useSecurity> to <useSecurity>false</useSecurity>

# Copy back to container
docker cp .\config.xml jenkins:/var/jenkins_home/config.xml

# Restart Jenkins
docker restart jenkins
7. Re-enable Security
Accessed Jenkins without login

Went to Manage Jenkins → Configure Global Security

Created new admin user

Restored security settings

Phase 4: Plugin Installation

Manage Jenkins → Manage Plugins → Available

Searched for "HTML Publisher"

Installed the plugin

Restarted Jenkins

Phase 5: Create and Run Pipeline
9. Create Pipeline Job
Clicked "New Item"

Named it: karunya-site-pipeline

Selected "Pipeline" type

Clicked OK

10. Configure Pipeline Script
Pasted this pipeline code:

groovy
pipeline {
    agent any
    stages {
        stage('Generate site') {
            steps {
                sh 'mkdir -p site'
                writeFile file: 'site/index.html', text: '''<html>
<head>
    <title>Karunya University</title>
</head>
<body>
    <h1>Karunya University — Simple Site</h1>
    <p>This site was generated by Jenkins.</p>
</body>
</html>'''
            }
        }
    }
    post {
        always {
            archiveArtifacts artifacts: 'site/**', fingerprint: true
        }
    }
}
11. Fix Syntax Error
Corrected HTML string by using triple quotes '''

Saved the pipeline configuration

Viewed Console Output for logs

Downloaded artifacts from Build Artifacts section

Opened index.html locally
